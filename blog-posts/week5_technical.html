Ruby Instance Methods and Variables

Written by Ronald Chacon on September 22, 2014.

Today, we will be going over the above example code. I am going to focus on the instance variable and instance method.

Instance Method's

In the above example. the class Person can be said to have 5 instance methods. attr_accessor provides 2 getter and 2 setter methods and the greet method is the last one. What this means, all instances of the class Person will know those methods.

Instance Variable's

All instance variables start with: @. Instance varialbles are available to the entire class instance. For example: p1 is instantiated with ("Bob", 42). And the greet method also has access to the instance variables. However if name and age did not have ampersand, they would be local variables, and the greet method would not have access to them.
